# Setup

## Create a config.json
```
{
    "Bot": {
        "Token": "your discord bot token",
        "Status": "3PG.xyz"
    },
    "DashboardURL": "http://localhost:3000",
    "LogSeverity": 4,
    "MongoURI": "mongodb://localhost/3PG-v1"
}
```
